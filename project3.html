담당 파트

3.캠핑존 예약 가능 여부 조회 및 예약 구현

  • 예약 여부 조회 시 예약 페이지의 전체가 아닌 일부분만 갱신하기 위해 ajax 사용
  • 예약 여부 조회 시 DatePicker를 사용하여 편리성 제공
  • 예약 여부 조회 시 다른 예약이 존재하는지 날짜 조회 쿼리를 통해 처리함

트러블 슈팅

1.캠핑장의 캠핑존(객실)들의 Vo Class List 출력 이슈

  • 이슈 : paging으로 처리 시 객실이 중복되어 출력.
  • 원인 : join으로 인한 paging 문제
  • 해결 : Stream을 사용하여 해결(단, 데이터 양이 많을 때 다른 방법을 고려해야할 것 같습니다.)

2.날짜로 캠핑장의 예약 여부 조회 시 쿼리문

  • 이슈 : 원하는 날짜에 특정 캠핑장의 특정 캠핑존(객실)의 예약 가능 여부 확인
  • 원인 : 예약 여부에 고려해야할 조건이 많았으며 특히 날짜 조건에서 시도를 많이 했습니다.
  • 해결 : not in과 예약 첫째날-마지막날의 범위를 고려하였습니다.
